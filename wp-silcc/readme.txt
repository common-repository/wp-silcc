WP-SiLCC differs from similar auto-tagging plugins because it utilizes active learning techniques to improve it's tagging ability over time. == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Request an API key from http://opensilcc.com ** You now no longer need an API key for normal use **
2. Download and Unzip wp-silcc
3. Upload the SiLCC plugin to plugin directory (../root/wp-content/plugins/)
4. Activate the plugin from the Wordpress admin panel
5. Enter your SiLCC API key
6. 1, 2, 3: You're done!
